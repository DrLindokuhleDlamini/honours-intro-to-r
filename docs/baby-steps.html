Getting Started with R

Dr Lindokuhle Dlamini
12 March 2026

1. Writing and Running a Script

Open RStudio and click:

File → New File → R Script

Steps:

  1. Write your code in the script editor
  2. Click Run or press Ctrl + Enter to execute a line of code

2. Understanding the RStudio Interface

Console

Where you type and execute commands.

Script Editor

Where you write and save multiple lines of code.

Environment

Stores variables and data.

Plots & Packages

Displays graphs and installed libraries.

3. Basic Commands

3.1 Arithmetic Operations

2 + 3   # Addition
5 - 2   # Subtraction
4 * 3   # Multiplication
10 / 2  # Division
2^3     # Exponentiation

3.2 Creating Variables

x <- 10   # Assigns 10 to x
y <- 5    # Assigns 5 to y
z <- x + y  # Adds x and y

z

3.3 Data Types

num <- 10.5
text <- "Hello"
bool <- TRUE

Check the data type

class(num)
class(text)
class(bool)

5. Creating and Using Data Frames

data <- data.frame(
  Name = c("Alice", "Bob", "Thandeka","Londeka","John","Zama"),
  Age = c(25,13,40,31,17,6)
)

data

data$Age

To access Thandeka’s age

data$Age[data$Name == "Thandeka"]

library(ggplot2)

9. Creating and Naming Objects in R

apples.sum <- 3 + 5

apples.sum

Assign names to values

John <- 3
Thandeka <- 5

apples.sum <- John + Thandeka

apples.sum

Naming Rules

Object names should:

• start with a letter
• contain letters, numbers, _ or .

Examples:

my_value <- 10
long_variable_name <- 2.5

10. Updating and Using Objects

apples.sum <- apples.sum * 2

apples.sum

11. Adding Comments

14. Summary

Key ideas:

• Use <- to assign values
• Use # to write comments
• Use functions to perform calculations

Scripts allow your analysis to be saved and reproduced later.
